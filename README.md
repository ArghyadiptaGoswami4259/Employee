# Employee Management System

A CRUD (Create, Read, Update, Delete) application for managing employee records, built and deployed with [Bolt.new](https://bolt.new).

## Live Demo

[https://optimize-performance-97dc.bolt.host/](https://optimize-performance-97dc.bolt.host/)

## Features

- Add new employee records
- View a list of all employees
- Update existing employee details
- Delete employee records
- Responsive, easy-to-use interface

## Tech Stack

- Built with [Bolt.new](https://bolt.new) (AI-assisted full-stack app builder)
- Frontend: React + TypeScript (Bolt.new default stack)
- Styling: Tailwind CSS
- Backend/Database: Supabase — *adjust if you're using a different backend*

## Getting Started

Since this project is built with Bolt.new, you can:

1. Open the project in [Bolt.new](https://bolt.new) to view and edit the source.
2. Use the **Deploy** feature in Bolt.new to publish changes.
3. Configure your database/backend (e.g. Supabase) with the required environment variables in the Bolt.new project settings.

### Running Locally (if exported)

```bash
git clone <your-repo-url>
cd employee-management
npm install
```
